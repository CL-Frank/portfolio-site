IT PAT 2022

Project Overview

The Backrooms is a short Unity-based horror game inspired by the popular internet concept. Players find themselves trapped in an endless maze of yellow rooms and must navigate through the liminal space to find an exit while being pursued by a terrifying monster.

Key Features

  • Immersive 3D maze environment with authentic Backrooms atmosphere
  • AI-driven monster with intelligent pathfinding and pursuit mechanics
  • Real-time score tracking system integrated with Azure database
  • Dynamic lighting and sound design for enhanced horror experience
  • Procedural maze generation for replayability

Technical Implementation

Built using Unity 3D engine with C# scripting, the game features advanced AI behavior trees for the monster entity, real-time database connectivity through Azure services, and optimized rendering for smooth gameplay performance.
