Project Overview

CampusLearn is a comprehensive educational platform that bridges the gap between traditional classroom learning and modern digital collaboration. Built with Spring Boot and designed with a mobile-first approach, the platform creates subject-specific communities where students can engage in meaningful academic discussions, access learning resources, and connect with qualified tutors. The system implements a robust role-based architecture supporting students, tutors, and administrators, each with tailored functionality to enhance the learning experience.

The platform addresses common challenges in university education by providing a centralized space for academic discourse, resource sharing, and peer-to-peer learning. Students can join communities based on their enrolled subjects, participate in threaded discussions, and access curated learning materials uploaded by tutors. The real-time messaging system facilitates direct communication between students and tutors, while the integrated AI chatbot provides instant academic assistance. Administrative features ensure content quality through comprehensive reporting and moderation systems.

Key Features

Technical Implementation

The application follows a layered architecture pattern with clear separation of concerns between the presentation, business, and data access layers. The backend leverages Spring Boot's dependency injection and auto-configuration capabilities to create a maintainable and scalable codebase. JPA entities with Hibernate ORM handle database operations, while custom repository interfaces provide optimized queries for complex data retrieval scenarios.

The frontend combines server-side rendering through Thymeleaf templates with client-side JavaScript for dynamic interactions. Bootstrap 5 provides the responsive framework, enhanced with custom CSS for brand consistency and mobile optimization. The WebSocket implementation uses STOMP protocol for real-time messaging, creating seamless communication channels between users without requiring page refreshes.

File storage integration with Cloudflare R2 (S3-compatible) enables scalable media handling for user-generated content including profile pictures, thread attachments, and learning resources. The system implements presigned URLs for secure file access and automatic cleanup procedures for deleted content to maintain storage efficiency.
